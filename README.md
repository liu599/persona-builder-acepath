# 🤖 PersonaBuilder - 首席人设架构师
PersonaBuilder 是 OpenClaw 核心技能组件，旨在解决大模型在执行专业任务时缺乏“上下文深度”和“性格逻辑”的问题。通过内置的 14 种顶级跨界人设及动态建人设 SOP，它可以瞬间将通用 AI 转化为极具专业深度的特定领域专家。

## 🌟 核心功能
- 内置 14 种顶级人设库：涵盖商业战略（马斯克、巴菲特）、技术架构（Python 后端架构师）、互联网增长（爆款写手、运营大拿）等全维度角色。
- 动态人设构建 (SOP)：支持通过 /建人设 指令或上传参考文档，分 3 步引导式创建专属人设。
- 5 大模块标准化输出：所有人设均包含身份设定、核心思维、质量标准、语气规范、绝对禁忌。
