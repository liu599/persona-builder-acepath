# 🤖 PersonaBuilder - 首席人设架构师
PersonaBuilder 是 OpenClaw 核心技能组件，旨在解决大模型在执行专业任务时缺乏“上下文深度”和“性格逻辑”的问题。通过内置的 14 种顶级跨界人设及动态建人设 SOP，它可以瞬间将通用 AI 转化为极具专业深度的特定领域专家。

## 🌟 核心功能
- 内置 14 种顶级人设库：涵盖商业战略（马斯克、巴菲特）、技术架构（Python 后端架构师）、互联网增长（爆款写手、运营大拿）等全维度角色。
- 动态人设构建 (SOP)：支持通过 /建人设 指令或上传参考文档，分 3 步引导式创建专属人设。
- 5 大模块标准化输出：所有人设均包含身份设定、核心思维、质量标准、语气规范、绝对禁忌。

## 🛠 使用指南
- 快速调用内置角色：直接输入角色名或 /人设库。
- 示例：“以史蒂夫·乔布斯的身份评价这个产品原型。”
- 动态定制人设：输入 /建人设，按照引导完成身份定位、思维逻辑和表达规范的设定。
- 文档解析：您可以直接上传职业简历、专家访谈或公司 SOP，AI 将自动提取特征并转化为 System Prompt。

## 📂 核心模块结构
所有人设均由以下五大核心模块约束，确保输出不“出戏”：

- Identity (身份设定): 确立坐标系与能力边界。
- Core Thinking (核心思维): 定义分析问题的底层引擎。
- Quality Standards (质量标准): 把控专业交付的底线。
- Tone (语气规范): 塑造沉浸式的外部皮肤。
- Absolute Taboos (绝对禁忌): 设置专业与行为护栏。
