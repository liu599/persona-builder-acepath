# 🤖 PersonaBuilder - Chief Persona Architect
PersonaBuilder is a core component of OpenClaw designed to solve the lack of "contextual depth" and "personality logic" in LLMs during professional tasks. By integrating 14 top-tier preset personas and a dynamic SOP workflow, it instantly transforms a general AI into a highly specialized expert in any field.

## 🌟 Key Features
- 14 Pre-installed Elite Personas: Covers business strategy (Elon Musk, Buffett), technical architecture (Python Backend Architect), internet growth (Copywriting Expert), and more.
- Dynamic Workflow (SOP): Create custom personas via the /建人设 command or document uploads through a guided 3-step process.
- 5-Module Standardized Output: Every persona is built upon Identity, Core Thinking, Quality Standards, Tone, and Absolute Taboos.

## 🛠 How to Use
- Invoke Preset Roles: Simply type a role name or /人设库.
- Example: "Critique this product prototype as Steve Jobs."
- Custom Persona Creation: Enter /建人设 and follow the prompts to define identity, logic, and communication style.
- Document Analysis: Upload resumes, expert interviews, or company SOPs; the AI will automatically extract traits and generate a System Prompt.

## 📂 Core Architecture
- To ensure high-fidelity outputs, all personas are governed by five pillars:
- Identity: Establishes the coordinate system and capability boundaries.
- Core Thinking: Defines the underlying engine for problem-solving.
- Quality Standards: Controls the baseline for professional delivery.
- Tone & Communication: Crafts the immersive "external skin" and voice.
- Absolute Taboos: Sets the guardrails for professional and behavioral limits.
